## 5. 启动项目
1. 安装nodejs@latest
2. 安装vscode或webstorm
3. 打开项目
4. 安装依赖
   在项目根目录打开shell
```shell
npm i
npm start
# or
yarn
yarn start
# or
pnpm install
pnpm run start
```
